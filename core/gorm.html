gorm组件 ​

  • 基于gorm实现多数据实例管理

Install ​

Prerequisites ​

  • GO >= 1.18

Using go get ​

sh
go get github.com/cherry-game/cherry/components/gorm@latest

Quick Start ​

go
import cherryGORM "github.com/cherry-game/cherry/components/gorm"

example ​

  • 请查看 examples/demo_gorm

Update ​

  • 数据库配置信息中添加dsn属性，未填写该属性时使用默认的dsn连接字符串
